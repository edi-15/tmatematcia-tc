CONCEPTOS

Números reales

Es la unión de los números racionales e irracionales.

Números racionales

Son todos los números que se pueden expresar en una fracción (decimales finitos, decimales periódicos y los decimales semiperiodicos).

Números irracionales

Son todos los números con infinitos decimales.

Poténcias

Es multiplicar el mismo número por sí mismo varias veces. El número que multiplicamos se llama base y el exponente es el número de veces que multiplicamos.

Raíces enésimas

Las raíces enésimas son lo que se multiplica n veces para tener el valor original

Multiplicación de raíces con igual índice

Es la multiplicación de las cantidades subradicales conservándose el índice que tienen en común.

División de raíces con igual índice

Es la división de las cantidades subradicales conservándose el índice que tiene en común.

Raíz de una raíz

Es la multiplicación de los índices de una raíz.

Racionalización

La racionalización consiste en quitar los radicales de un dominador, para hacerlas más fáciles.

Logaritmos

El logaritmo de un número, en una base dada, es el exponente al cual se debe elevar la base para obtener el número.
